# Witchcraft Daily Practice Discord Bot

Discord bot that delivers daily witchcraft practice prompts, moon phase reminders, tarot card of the day, herb and crystal correspondence tables, spell timing recommendations based on planetary hours, and sabbat countdowns. Commands include /moon for current lunar phase, /tarot for daily draw with interpretation, /herb for random correspondences, /spell-timing for auspicious windows, /sabbat for upcoming wheel of the year events, and /daily-practice for personalized morning ritual suggestions. Built for witchcraft communities with soft mystical voice. Configurable timezone, guild-specific settings, and optional role-based premium features. Self-hostable with Docker, full slash command support, Discord.py, MIT license.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
